# blackwood

Blackwood Hollow es un escenario completamente ficticio creado para una partida de rol de Mage: The Ascension (20th Anniversary Edition). Este repositorio contiene el código fuente y los contenidos de un sitio web dentro del universo del juego, diseñado para simular un recurso informativo.

Contenido ficticio:
- Todo el contenido —lugares, organizaciones (incluyendo ART), eventos y personajes— es ficción creada para la campaña de rol.
